# Contributing to CommuteCompute

Thank you for your interest in contributing to CommuteCompute.

To protect the integrity and ownership of the project, contributions are
accepted only under the conditions outlined below.

## Scope of Contributions

Contributions are welcome **only** in the following areas:

- Documentation
- User interface components
- Configuration files
- Build, tooling, and deployment scripts
- Non-decision-making integrations or adapters

## Restricted Areas

Contributions are **not accepted** for:

- Core decision-intelligence logic
- Optimisation or weighting algorithms
- Behavioural inference systems
- Any component designated as proprietary or trade secret

Pull requests affecting these areas will be declined.

## Intellectual Property

By submitting a contribution, you agree that:

- Your contribution is original work
- You grant the project owner a perpetual, irrevocable license
  to use, modify, and distribute the contribution
- No ownership or authorship rights to the CommuteCompute core are implied

## No Implied Partnership

Contributing to this repository does not imply employment, partnership,
or ownership interest in CommuteCompute.

Thank you for helping improve the project responsibly.
